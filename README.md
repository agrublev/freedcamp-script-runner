# ./bin

## Install

```sh
$ npm i ./bin -g
```

## Usage

```sh
$ ./bin --help
```

Help output:

```
[2J[0f19:18:59 [LOG]Ô∏è Usage: bin <command> [options]

Commands:

  bin scripts     Choose a script from package.json
  bin list        Select any task with text autocompletion

                  'ignore-upgrade':[]

                  file
```
